    #### ■ point 
    - while文やdo-while文では、中括弧を省略した場合、次の文だけが繰り返しの対象になる。  
    - do-while文で中括弧を省略した場合には、doとwhileの間には一文のみを記述できる。二文以上記述した場合にはコンパイルエラーになる。  
    - セミコロンが現れるまでが一つの文のため,中カッコを使わないwhile文の中に複数行に渡るdo-while文を記述することは問題ない。  
  ``` java
    while(条件文)
      do{
        //複数行の処理
        
      }while;
  ```
